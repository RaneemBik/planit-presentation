# PlanIt — Presentation Slides

A polished React-based presentation for the PlanIt project. Navy & white color scheme with smooth animations.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Navigation

- **Arrow keys** ← → to navigate slides
- **Click** nav dots at the top to jump to any slide
- **Click** slide names at the bottom for quick access
- **→ Button** (bottom right) to advance

## Slides

1. Introduction & Project Overview
2. Target Users
3. Tech Stack & Technologies
4. Main Website Flow
5. Core Features & Functionalities
6. AI Features
7. Additional Functionalities
8. Challenges & Solutions

## Build for Production

```bash
npm run build
npm run preview
```
